public string
# generateUrl( string $repositoryPath, null $currentUrl = null )

Generates a URL for a Puli asset.

Generates a URL for a Puli asset.

Depending on the configuration of the resource, the returned URL is either relative to the current domain or absolute.

If you pass the current URL, the returned URL is relative to the passed URL.

Parameters

$repositoryPath
The repository path of the resource.
$currentUrl
The current URL.

Returns

string
The URL of the resource.

Throws

Puli\UrlGenerator\Api\CannotGenerateUrlException
If the URL cannot be generated.
